Methodology & Assumptions

What this calculates: An adjusted lot size that scales a base position according to win rate, risk-reward ratio, and reference volatility. This is a planning signal, not a recommendation.

Formula (heuristic): Adjusted Lot = Base Lot × (Win Rate / 50) × (RR / 2) × (200 / Volatility)

Note — this is a heuristic formula. It provides a directional sizing signal based on simplified assumptions and should not be treated as a statistically validated model. The scaling factors (50, 2, 200) are reference anchors, not calibrated constants.

Limitations: Does not account for correlation between strategy assumptions, non-normal return distributions, or regime changes in volatility. Use alongside other planning tools for a fuller risk picture.
