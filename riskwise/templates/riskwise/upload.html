What happens next

Once your file is loaded, follow the product in this order.

1. Build planning baseline: RiskWise reads the uploaded outcome data and prepares a baseline view of downside depth, loss severity, and edge quality.
2. Review downside and stability signals: Open the dashboard to evaluate observed drawdown pressure, volatility, and historical efficiency before taking new risk.
3. Stress-test and compare: Run simulations and compare alternative planning assumptions before committing capital.
Accepted file guidance
  • CSV and Excel (.xlsx/.xls) files are supported
  • A profit column is required for simulation workflows
  • Date fields improve filtering and provenance context
  • Use realistic historical outcomes for more meaningful planning signals
Import from TradeIntel 360
  • Current path: manual CSV/XLSX/XLS upload
  • Required outcome field: profit or equivalent P&L naming
  • Helpful context: date/time, session, symbol, side, and status fields
  • No live sync or direct API pull in the current public release
